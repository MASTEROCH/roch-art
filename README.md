# ROCH Landing

Лендинг услуг ROCH для roch-audit.com. Один файл, ноль зависимостей.

## Запуск
Открой `index.html` в браузере. Для деплоя — положи в корень проекта
как `start.html` → откроется на roch-audit.com/start.

## Файлы
- `index.html` — сам лендинг (переименуй в start.html при деплое)
- `О-ПРОЕКТЕ-цель.md` — назначение и стратегия
- `СТИЛЬ-design-system.md` — визуальный язык (для работы в VS Code)
- `PITCH-как-презентовать.md` — шпаргалка на ивент

## Перед запуском
1. Проверь Telegram-хендл: `var TG_USER = 'broch'` в начале `<script>`
2. Проверь реквизиты в блоке оплаты (id `wallet`, `iban`)
3. Открой по живой ссылке в Safari (не в Quick Look — он режет стекло)
